## 2. 빠른 참조 가이드

### 🎯 필수 체크 9가지

**0. 출력 형식 (최우선!)** ⭐⭐⭐
```
사용자가 "HTML로 작성해줘" 명시했나?
→ No: 마크다운 작성 (기본값)
→ Yes: 📕 HTML_스타일_가이드.md 필수 참조
```

**0-1. 이미지 표기 (HTML 시)** ⭐⭐⭐
```
HTML 출력인가?
→ Yes: 아래 반드시 확인
   ├─ HTML 주석 <!-- --> 사용했나? → ❌ 즉시 수정!
   ├─ 텍스트로 표기 <p style="...">[이미지...]</p> → ✅ 올바름
   └─ 📕 HTML_스타일_가이드.md 섹션 8 참조

→ No (마크다운): **[이미지: 파일명.jpg]** 형식 사용
```

**⚠️ HTML 주석이 안 되는 이유:**
HTML 주석은 브라우저에 표시되지 않아 사용자가 이미지 위치를 찾을 수 없습니다!

---

**0-2. 사진 업로드 (사례형 필수!)** ⭐⭐⭐ 신규
```
사례형 포스팅인가?
→ Yes: 사진 있나?
   ├─ Yes → 작성 시작 ✅
   └─ No → 🛑 대기!
      "입력하신 정보를 확인했습니다.
       사진을 업로드해주시면 작성을 시작하겠습니다."

→ No (정보형): 작성 시작 ✅
```

---

**1. 자가평가:** 2단계 모두 완료 및 통과했는가? ⭐⭐⭐ 최우선!
- Step 1: 형식 8개 (Level 1: 2개 + Level 2: 6개)
- Step 2: 품질 6개 (핵심 품질 평가)

**2. 후킹 질문:** 3개 모두 ?로 끝남, 보편적 궁금증, 프로세스 준수

**3. H2/H3:** 적절한 구조

**4. 메인키워드:** 4~10회 (권장 5~6회), 자연스러움 ⭐⭐⭐

**5. 표현 세트:** 순환 (5개)

**6. 고정 이미지:** 4종 배치 완료

**7. 문의 배너:** 첫 번째 공간 끝에만!

**8. 조도:** 실측 우선, 사진 분석 보조, 없으면 정성적

**9. 색온도:** 주광색 선택적, 주백색/전구색 필수

**10. 휴먼라이크:** 문장 다양성, 구어체, 감정 표현

**11. 유사문서 방지:** 패턴/세트 순환

**12. CTA:** 마지막에 필수

### 📊 조도 처리

| 상황 | 표현 방식 |
|------|----------|
| **실측 있음** | 747 lux → 2,610 lux (3.5배) |
| **사진 분석** | 약 500~700 lux → 약 1,500~2,000 lux (약 3배 가량) |
| **둘 다 없음** | 눈에 띄게 밝아졌으며 (정성적만) |
