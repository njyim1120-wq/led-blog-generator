## 1. LED 수명의 진실

### ❌ 잘못된 정보

```
"LED는 50,000시간 수명이라 10년 사용 가능합니다!"
"LED는 반영구적입니다!"
"10년 무걱정입니다!"
"한 번 설치하면 10년 이상 사용 가능합니다!"
```

→ **완전히 잘못된 정보!**

---

### ✅ 정확한 정보

#### **LED칩 vs LED 조명**

**LED칩 (발광다이오드):**
- 수명: 50,000시간 ✅

**LED 조명 (실제 제품):**
- LED칩을 활용한 모듈
- 컨버터(안정기/SMPS) 연결
- **실제 조명 수명 = 기판(모듈) + 안정기(SMPS) 성능에 따라 결정**

---

#### **왜 2년 만에 고장 나나요?**

**고객 불만:**
```
"LED가 반영구적이고 10년 이상 사용할 수 있다고 했는데,
왜 이렇게 2년밖에 안 됐는데 쉽게 고장이 나죠?"
```

**이유:**
```
LED칩 수명 ≠ LED 조명 수명

LED 조명 = LED칩 + 기판(모듈) + 안정기(SMPS)
→ 기판과 안정기 성능에 따라 실제 수명 결정
→ 2~3년 내 고장 발생 가능 (정상)
```

---

#### **휴명의 대응**

**제품 선별:**
```
✅ 검증된 제조사 제품 사용
✅ 불량이 적은 제품 선별
✅ 그럼에도 불량 발생 가능 (완전 방지 불가)
```

**AS 정책:**
```
✅ 2년 AS 보증
✅ 불량 발생 시 즉시 대응
✅ 고객에게 정확한 정보 제공
```

**고객 이해:**
```
LED 수명에 대한 정확한 정보 제공
    ↓
고객들이 이해해줌
    ↓
AS 정책으로 신뢰 구축
```

## 5. 고객 질문 대응법

### 📌 FAQ 작성 시 주의

**질문: "LED는 얼마나 오래 사용할 수 있나요?"**

**❌ 잘못된 답변:**
```
"LED는 50,000시간 수명이라 하루 8시간 사용 기준으로 
약 10년 이상 사용 가능합니다."
```

**✅ 올바른 답변:**
```
"LED칩 자체는 50,000시간의 수명을 가지고 있지만, 
실제 LED 조명의 수명은 기판과 안정기 성능에 따라 달라집니다.
저희는 검증된 제품을 사용하여 불량률을 최소화하고 있으며, 
2년 AS 보증으로 안심하고 사용하실 수 있습니다."
```

---

**질문: "LED도 고장 나나요?"**

**❌ 잘못된 답변:**
```
"LED는 거의 고장이 안 나고 반영구적으로 사용 가능합니다."
```

**✅ 올바른 답변:**
```
"LED 조명도 전자제품이므로 불량이나 고장이 발생할 수 있습니다.
다만 저희는 검증된 제품을 선별하여 불량률을 최소화하고 있으며,
만약의 경우에도 2년 AS로 책임지고 관리해드립니다."
```

---

**질문: "형광등보다 수명이 긴가요?"**

**❌ 잘못된 답변:**
```
"네, LED는 형광등보다 훨씬 수명이 길어서 10년 이상 사용 가능합니다."
```

**✅ 올바른 답변:**
```
"일반적으로 LED가 형광등보다 수명이 긴 것은 맞습니다.
하지만 실제 제품 수명은 사용 환경과 제품 품질에 따라 달라지므로,
저희는 2년 AS 보증으로 품질을 책임지고 있습니다."
```
