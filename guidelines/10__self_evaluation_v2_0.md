## 3. Level 1: 치명적 형식 (2개)

### ⚠️ 하나라도 오류면 즉시 재작성!

---

### 1️⃣ LED 수명 금지표현 ⭐⭐⭐

**왜 치명적인가:**
```
LED 수명 과장 → 고객 클레임 → 회사 신뢰도 추락
```

**금지 표현 6가지:**
```
❌ "10년"
❌ "반영구"
❌ "50,000시간"
❌ "평생"
❌ "무걱정"
❌ "거의 고장 안 남"
```

**체크 방법:**
```
포스팅 전체 검색:
- "10년" → 발견?
- "반영구" → 발견?
- "50,000시간" → 발견?
- "평생" → 발견?
- "무걱정" → 발견?
- "거의 고장" → 발견?

→ 하나라도 발견 → 즉시 수정
```

**수정안 제시:**
```
[현재] "LED는 10년 무걱정으로 사용하실 수 있습니다."
[수정] "2년 AS 보증으로 안심하고 사용하실 수 있습니다."

[현재] "반영구적으로 사용 가능합니다."
[수정] "검증된 제품으로 불량률을 최소화했습니다."
```

**✅ 통과 기준:**
- 금지 표현 6개 모두 없음
- "2년 AS" 2곳 배치됨

---

### 2️⃣ 후킹 질문 물음표 ⭐⭐⭐

**왜 치명적인가:**
```
물음표 없음 → 후킹 질문 아님 → 독자 유입 실패
```

**체크 방법:**
```
H2 제목 3개 확인:
- 1번 질문: "?"로 끝나는가?
- 2번 질문: "?"로 끝나는가?
- 3번 질문: "?"로 끝나는가?

→ 하나라도 평서문 → 즉시 수정
```

**❌ 잘못된 예:**
```
## LED로 바꾸면 밝아집니다. ← 평서문!
## 전력을 절감할 수 있습니다. ← 평서문!
```

**✅ 올바른 예:**
```
## LED로 바꾸면 얼마나 밝아질까요?
## 전력을 얼마나 절감할 수 있나요?
```

**수정안 제시:**
```
[현재] "LED로 바꾸면 밝아집니다."
[수정] "LED로 바꾸면 얼마나 밝아질까요?"

[현재] "전력을 절감할 수 있습니다."
[수정] "전력을 얼마나 절감할 수 있나요?"
```

**✅ 통과 기준:**
- 3개 모두 "?"로 끝남
- 50자 이내

## 4. Level 2: 필수 형식 (6개)

### ⚠️ 하나라도 누락되면 추가 필요!

---

### 3️⃣ 사진 업로드 (사례형만) ⭐⭐⭐

**왜 필수인가:**
```
사례형 = 실제 시공 사례
→ 사진 없으면 신뢰도 0
```

**체크 방법:**
```
포스팅 타입 확인:
- 사례형인가?
  → Yes: 사진 체크
    - 설치 전 사진 있는가?
    - 작업 중 사진 있는가?
    - 설치 후 사진 있는가?
  → No: 통과
```

**필수 구성:**
```
✅ 설치 전: 최소 1장
✅ 작업 중: 최소 2장
✅ 설치 후: 최소 2장
```

**✅ 통과 기준:**
- 사례형인데 사진 없음 → 작성 불가
- 정보형 → 통과
- 사례형 + 사진 있음 → 통과

---

### 4️⃣ 고정 이미지 4종 배치 ⭐⭐

**필수 배치:**

**정보형 (3종):**
```
1. [GIF: 업체소개.gif]
2. [썸네일: 대표이미지.jpg]
3. [GIF: 조도비교.gif] (선택적)
```

**사례형 (4종):**
```
1. [GIF: 업체소개.gif]
2. [썸네일: 대표이미지.jpg]
3. [GIF: 거실_조도비교.gif]
4. [배너: 문의배너.jpg] - 첫 공간만!
```

**체크 방법:**
```
텍스트 검색:
- "[GIF: 업체소개.gif]" → 발견?
- "[썸네일: 대표이미지.jpg]" → 발견?
- "[GIF:" → 조도비교 발견?
- "[배너: 문의배너.jpg]" → 사례형만 발견?
```

**✅ 통과 기준:**
- 정보형: 최소 3종
- 사례형: 4종 모두

---

### 5️⃣ LED 수명 배치 2곳 ⭐⭐

**필수 배치 위치:**
```
1. 시공 개요 - 소비전력 표 다음

```

**체크 방법:**
```
텍스트 검색:
- "2년 AS" 또는 "2년 무상 AS"
- 발견 횟수: 2회?

위치 확인:
- 첫 번째: 소비전력 표 다음?
- 두 번째: 포스팅 끝?
```

**✅ 통과 기준:**
- 정확히 2곳
- 위치 정확

## 5. Level 3: 핵심 품질 (6개) ⭐ 신규

### 품질 1️⃣ 후킹 질문 품질 ⭐⭐⭐

**평가 항목:**

**[1] 프로세스 준수 (최우선)**
```
✅ 입력 정보 분석 → 글의 주제 파악
✅ 주제의 보편적 궁금증 10개 이상 도출
✅ 글에서 답할 수 있는 것 3개 선택
✅ 질문으로 작성 (?, 50자 이내)
```

**[2] 답변 가능성**
```
각 질문마다:
✅ 이 글에서 실제로 답하는가?
✅ 어느 섹션에서 답하는가?
✅ 답변이 충분히 구체적인가?
```

**[3] 보편성**
```
✅ 특정 사례가 아닌가?
✅ 대부분의 독자가 공감할 수 있는가?
✅ "내가 궁금했던 거!" 느낌을 주는가?
```

**평가 출력 예시:**
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[1/6] 후킹 질문 품질 평가...
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

입력 정보 분석:
- 디밍안정기 고장
- 디밍스위치 교체 고민
- LED 리폼

↓

글의 주제 파악:
"디밍스위치 교체 없이 LED 리폼으로 해결"

↓

보편적 궁금증 도출 확인:
1. 디밍안정기 고장 시 디밍스위치 꼭 교체?
2. 디밍스위치 그대로 사용하면서 LED 가능?
3. 수리 vs 리폼 경제성?

✅ 프로세스 준수: 완료

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
각 질문 검증:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

질문 1: "디밍안정기가 고장 나서 형광등이 안 들어오는데 
         꼭 디밍스위치를 교체해야 하나요?"

✅ 답변 가능: 회사 레퍼런스 + 마무리에서 답함
✅ 보편적 궁금증: 디밍안정기 고장 공통 고민
✅ 특정 사례 아님

질문 2: "디밍스위치를 그대로 사용하면서 
         LED로 바꿀 수 있는 방법은 없을까요?"

✅ 답변 가능: 회사 레퍼런스 + 설치 후에서 답함
✅ 보편적 궁금증: 디밍스위치 아파트 공통
✅ 독립적 질문

질문 3: "디밍안정기 수리 비용보다 LED 리폼이 
         더 경제적이라는데 정말인가요?"

✅ 답변 가능: 소비전력 비교 + 마무리에서 답함
✅ 보편적 궁금증: 경제성 비교
✅ 독립적 질문

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 후킹 질문 품질: 우수

```

**[1] 횟수 적정성**
```
✅ 4~10회 범위 (권장 5~6회)
✅ 3회 이하: 부족
✅ 11~19회: 약간 많음 (통과)
✅ 20회 이상: 키워드 스터핑 (패널티)
```

**[2] 자연스러움**

## 9. 빠른 참조

### 📊 3단계 평가 구조

```
Level 1: 치명적 형식 (2개) → 즉시 재작성
  1. LED 수명 금지표현
  2. 후킹 질문 물음표

Level 2: 필수 형식 (6개) → 추가 필요
  3. 사진 업로드 (사례형)
  4. 고정 이미지 4종
  5. LED 수명 배치 2곳
  6. 회사 레퍼런스 2문단
  7. CTA 존재
  8. H3 구조 (사례형)

Level 3: 핵심 품질 (6개) → 개선 제안
  1. 후킹 질문 품질
  2. 메인키워드 자연스러움
  3. 시공 프로세스 정확성
  4. 광고/정보 균형
  5. 현장감 & 공감
  6. 지침 준수 종합
```

---

### 🎯 핵심 원칙

```
1. Level 1 통과 → Level 2
2. Level 2 통과 → Level 3
3. Level 3 우수 → 제출 가능

단, Level 1 오류 → 즉시 중단, 재작성
```
