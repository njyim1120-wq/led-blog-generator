## 1. 시공 유형 개요

### 1-1. LED 리폼이란?

**LED 리폼 (LED Reform):**
- **프레임(등기구 구조) 유지** ← 핵심!
- 안정기 + 광원만 교체
- 비용 절감 (30~50%)

---

### 1-2. LED 리폼 vs 등기구 전체 교체

#### 📊 비교표

| 구분 | LED 리폼 | 등기구 전체 교체 |
|------|---------|---------------|
| **교체 범위** | 안정기 + 광원만 | 등기구 전체 (프레임 포함) |
| **프레임** | 유지 ✅ | 교체 |
| **비용** | 낮음 (30~50% 절감) | 높음 |
| **시공 시간** | 짧음 (1~2시간) | 김 (반나절~하루) |
| **전력 절감** | 50~60% | 50~60% (동일) |
| **밝기 개선** | 3배 이상 | 3배 이상 (동일) |
| **적용 가능** | 프레임 상태 양호 | 프레임 노후/손상 |

---

#### 🔧 언제 리폼?

**적용 상황:**
```
✅ 등기구 프레임 상태 양호
✅ 비용 절감 중요
✅ 빠른 시공 원함
✅ 현재 디자인 유지
```

**대표 케이스:**
- 거실등 LED 모듈 교체
- 주방등 형광등 → LED
- 방등 LED → LED (성능 개선)

---

#### 🔧 언제 전체 교체?

**적용 상황:**
```
✅ 프레임 노후화/손상
✅ 디자인 변경 원함
✅ 바리솔 천장 교체
✅ 완전히 새롭게 시작
```

**대표 케이스:**
- 바리솔 처짐 → 전면 교체
- 오래된 등기구 → 새 디자인
- 프레임 손상 → 신규 설치

---

### 1-3. 실링팬 설치 개요

#### 🌀 실링팬이란?

**실링팬 (Ceiling Fan):**
- 천장 설치형 선풍기 + 조명
- 공기 순환 + 인테리어 효과
- 전력 효율적 (에어컨 대비)

---

#### 📊 실링팬 vs 일반 선풍기

| 구분 | 실링팬 | 일반 선풍기 |
|------|--------|----------|
| **설치 위치** | 천장 고정 | 바닥/테이블 |
| **공기 순환** | 공간 전체 ✅ | 국소적 |
| **인테리어** | 세련된 ✅ | 일반적 |
| **전력 소비** | 30~60W | 40~80W |
| **조명 일체** | 가능 ✅ | 불가능 |
| **설치 비용** | 높음 | 낮음 |

---

#### 🔧 실링팬 설치 이유

**고객이 실링팬을 선택하는 이유:**
```
1. 공기 순환 효과 (에어컨 보조)
2. 전력 절감 (에어컨 대비 1/10)
3. 세련된 인테리어
4. 조명 + 선풍기 일체형
5. 사계절 사용 가능
```

## 2. 시공 프로세스

### 2-A. LED 리폼 프로세스 ⭐ 수정

#### 📋 전체 흐름

```
1. 전원 차단 (필수!)
2. 확산판(커버) 분리
3. 기존 광원 제거
4. 기존 안정기 제거 ⭐ 디밍 기능 주의
5. 프레임 청소 (프레임 유지!)
6. 전선 상태 확인 및 처리
7. 새 LED 컨버터 장착
8. 새 LED 모듈 장착 (자석 방식)
9. 점등 테스트
10. 확산판 재조립
```

---

#### 🔧 단계별 상세

**1단계: 전원 차단**
- 차단기 내림 (필수!)
- 안전 확보

**2단계: 확산판(커버) 분리**
- 확산판 또는 아크릴 커버 분리
- 내부 형광등/LED 노출
- 나사/클립 잘 보관

**3단계: 기존 광원 제거**

**형광등의 경우:**
- 형광등 탈거 (55W 또는 36W)

**기존 LED의 경우:**
- 고정 나사 제거
- 커넥터 분리
- LED 모듈 제거

---

**4단계: 기존 안정기 제거** ⭐⭐⭐ 매우 중요!

⚠️ **일반 형광등의 경우:**
```
형광등 = 형광등 + 안정기 (1:1 구조)
55W 형광등 1개 → 55W 안정기 1개

LED 리폼 시:
→ 기존 안정기 제거 (필수!)
→ 안정기 전선 분리
→ 안정기 고정 나사 제거
→ 단자만 남김
```

---

⚠️ **디밍 스위치의 경우:** ⭐⭐⭐ 신규 추가

```
형광등 = 형광등 + 디밍안정기 (1:1 구조)

LED 리폼 시:
→ 디밍안정기 제거 (필수!)
→ LED 전용안정기(SMPS) 설치
→ ⭐⭐⭐ 중요: 디밍 기능(밝기 조절) 작동 안 함
→ 온/오프만 작동 (켜기/끄기만 가능)
→ 스위치는 그대로 유지 (교체 불필요)
→ 벽면 공사 없음 (비용 절감)
```

**디밍스위치 + LED 설치의 핵심:**
```
✅ 스위치 교체 없음
✅ 벽면 공사 없음
✅ 비용 절감
⚠️ 밝기 조절 기능 작동 안 함 (온오프만)
```

**적용 아파트:**
- 퍼스트월드
- 엑스포 6, 9, 10단지
- 센트럴파크 1차, 2차
- 웰카운티 1, 2, 3, 4단지

---

**5단계: 프레임 청소**

⚠️ **리폼 핵심!**
```
✅ 프레임은 제거하지 않고 유지함
✅ 먼지, 이물질 제거
✅ 프레임 상태 확인
```

**6단계: 전선 상태 확인 및 처리**

**점검 항목:**
- 단자 상태 확인
- 배선 상태 확인
- 배선 노후/손상 여부

**전선 처리:**
```
상황 1: 전선이 낡거나 삭은 경우
→ 조치 1: 해당 부분 절단
→ 조치 2: 절연테이프로 보강
→ 조치 3: 필요 시 새 배선 추가

상황 2: 단자가 낡거나 삭은 경우
→ 조치: 단자 교체
```

**7단계: 새 LED 컨버터(전용안정기/SMPS) 장착**
- LED 전용안정기(SMPS) 1개 설치
- 배선 연결
- 고정

**핵심:**
```
기존 안정기 2~3개 → LED 전용안정기 1개
```

**8단계: 새 LED 모듈 장착 (자석 방식)**

⚠️ **최신 방식!**

**방법 1: 자석 부착 (최신 방식)**
```
LED 모듈 뒷면에 자석
등기구 철판에 부착

장점:
- 탈착 간편
- AS 용이
- 빠르고 정확한 위치 고정
```

**방법 2: 나사 고정 (기존 방식)**
```
나사로 모듈 고정

장점:
- 안정적 고정
```

**연결:**
- 커넥터 연결
- 극성 확인 (+ -)
- 커넥터 확실히 연결

**9단계: 점등 테스트**
```
전원 ON
모든 LED 점등 확인
밝기 확인
디밍 작동 확인 (해당 시) → ⚠️ 디밍스위치는 온오프만!
깜박임 확인

이상 발견 시:
- 커넥터 재연결
- 모듈 교체
- 배선 점검
```

**10단계: 확산판 재조립**
- 등커버(확산판) 닫기
- 리폼 완성! ✅

## 3. 기본 치환 규칙

### 📊 형광등 → LED 치환표

| 기존 형광등 | LED 모듈 | 절감 전력 |
|------------|---------|----------|
| 55W | 25W | 30W |
| 36W | 18W | 18W |

---

### 🔍 적용 원칙

1. **형광등 종류 확인:** PL 55W or 36W
2. **개수 확인:** 2등/3등/1등
3. **계산:**
   - 기존 전력 = 형광등 W × 개수
   - 교체 후 = LED W × 개수
   - 절감 = 기존 - 교체 후

## 4. 안정기 구조

### 📌 형광등 시스템 구조

**형광등 = 형광등 + 안정기 (1:1 구조)**
```
형광등 1개 → 안정기 1개
형광등 2개 → 안정기 2개
형광등 3개 → 안정기 3개
```

---

### 🔧 LED 리폼 후 구조

**거실 2등 (LED):**
```
LED 모듈 25W × 2개
    +
LED 전용안정기(SMPS) 50W 1개

핵심: 안정기 2개 → LED 전용안정기 1개
```

**방등 (LED):**
```
LED 모듈 18W × 3개 (또는 25W × 2개)
    +
LED 전용안정기(SMPS) 1개

핵심: 안정기 3개 → LED 전용안정기 1개
```

---

### ⚠️ 중요: 안정기 제거 필수!

**형광등 → LED 리폼 시:**
```
❌ 기존 안정기 그대로 두면 안 됨!
✅ 기존 안정기 제거 (필수!)
✅ LED 전용안정기(SMPS) 1개만 설치
```

**이유:**
- 형광등 안정기는 LED와 호환 안 됨
- LED는 LED 전용안정기(SMPS) 필요

## 💡 실전 팁

### Tip 1: 시공 유형 구분

**사진 분석 시:**
```
프레임 상태 양호 → LED 리폼
프레임 노후/손상 → 전체 교체
바리솔 처짐 → 전체 교체
선풍기+조명 원함 → 실링팬 설치
```

---

### Tip 2: 포스팅 표현

**LED 리폼:**
```
"LED 모듈을 교체하여..."
"기존 등기구 틀은 그대로 두고..."
```

**전체 교체:**
```
"새 등기구를 설치하여..."
"등기구 전체를 교체하여..."
```

**실링팬:**
```
"실링팬을 설치하여..."
"공기 순환과 인테리어 효과를..."
```

**디밍스위치:** ⭐ 신규
```
"스위치 교체 없이..."
"추가 공사 없이..."
"기존 디밍스위치를 그대로..."
```

---

### Tip 3: 계산 검증

**절감률 체크:**
- 형광등 → LED: 보통 50~60% 절감
- LED → LED: 보통 40~50% 절감
- 실링팬: 50~60% 절감 (기존 형광등 대비)
- 너무 높거나 낮으면 재확인

## 📌 빠른 참조

### 시공 유형 3가지

```
A. LED 리폼: 프레임 유지, 안정기+광원 교체
B. 전체 교체: 프레임 포함 전체 교체
C. 실링팬: 천장 보강 + 실링팬 설치
```

---

### 기본 치환 규칙

```
55W 형광등 → 25W LED
36W 형광등 → 18W LED
```

---

### 공간별 구조

```
거실: 2등 or 3등
주방: 1등 or 2등 (밝기 보강 가능)
방: 사각 or 원형
실링팬: 매입등 + 모터
```

---

### 실링팬 소비전력

```
매입등: 10~25W (일반 20W)
모터: 20~40W (일반 30W)
합계: 30~65W (일반 50W)
```

---

### 디밍스위치 + LED 설치 ⭐ 신규

```
✅ 스위치 교체 없음
✅ 벽면 공사 없음
✅ 비용 절감
⚠️ 온/오프만 작동 (밝기 조절 안 됨)

적용 아파트:
- 퍼스트월드
- 엑스포 6, 9, 10단지
- 센트럴파크 1차, 2차
- 웰카운티 1, 2, 3, 4단지
```

---

### 포스팅 원칙

```
기술 상세 ❌
결과 중심 ✅
자연스럽게 ✅
디밍 기능 표현 ❌ ⭐
```
