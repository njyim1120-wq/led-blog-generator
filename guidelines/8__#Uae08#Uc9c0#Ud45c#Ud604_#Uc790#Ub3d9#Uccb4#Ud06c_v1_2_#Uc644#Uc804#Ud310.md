## 3. 키워드 20회 제한 (이 문서의 주 목적) ⭐

### 📊 키워드 스터핑 방지

**네이버 SEO 규칙:**
- 메인키워드 **권장: 5~6회**
- 메인키워드 **적정: 4~10회**
- 메인키워드 **20회 초과 → 키워드 스터핑** ⭐
- 패널티 가능

**자가평가에서 체크되는 것:**
- 4~10회 범위 체크
- 범위 벗어나면 자동 수정

**이 문서에서 체크하는 것:**
- **20회 초과 여부** ⭐
- SEO 패널티 위험 확인

---

### 🔍 Ctrl+F 검색

```
Ctrl+F로 메인키워드 검색:

예시: "송도LED조명"
  ↓
검색 결과: 6/6 (6개 발견)
  ↓
판정: ✅ 통과 (5~10회 적정)
```

---

### ⚠️ 20회 초과 시 (치명적!)

**발견 시:**
```
❌ 23/23 (23개 발견) ← 키워드 스터핑!
  ↓
즉시 수정: 일부 키워드를 자연스럽게 제거
  ↓
목표: 15회 이하로 줄이기
```

**예시:**
```
수정 전: "송도LED조명 전문 업체 휴명은 송도LED조명 시공에..."
수정 후: "저희는 조명 전문 업체로서 송도에서..."
```

---

### 📝 키워드 사용 원칙

**권장 배치:**
```
제목: 1회 (필수)
서론: 1~2회

결론: 1회
━━━━━━━━━━━

최대: 10회 (허용)

```

## 4. HTML 작성 시 주의 (추가 체크)

### ⚠️ HTML 주석 금지!

**절대 금지:**
```
❌ <!-- [이미지: 파일명.jpg] -->
❌ <!-- [GIF: 업체소개.gif] -->
❌ <!-- [배너: 문의배너.jpg] -->
```

**이유:**
- HTML 주석은 화면에 표시 안 됨
- 사용자가 이미지 위치 찾을 수 없음

---

### 🔍 Ctrl+F 검색

```
Ctrl+F로 "<!--" 검색:
  ↓
발견됨? → ❌ 즉시 수정!
없음? → ✅ 통과
```

---

### ✅ 올바른 방법

```
<p style="text-align: center; color: #9c27b0;">
[이미지: 파일명.jpg]
</p>
```

**📚 상세:** HTML_스타일_가이드.md
