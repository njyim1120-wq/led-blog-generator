## 1. v3.0 핵심 변경사항

### ❌ v2.0의 문제

```
입력 변수를 직접 질문으로 변환
→ 특정 사례, 보편성 없음
→ 억지 스토리 구조 (공감-불편-해결)
```

### ✅ v3.0의 접근

```
입력 정보 → 글의 주제 파악
→ 그 주제의 보편적 궁금증 도출
→ 독립적 질문 3개 선택
```

**핵심:**
- 네이버 블로그 = 검색 기반 = 궁금증 해결
- "아 맞아, 내가 궁금했던 거!"

## 2. 작성 프로세스 (4단계)

### Step 1: 글의 주제 파악

```
입력: 디밍스위치, 형광등 → LED, 리폼
주제: "디밍스위치 아파트에서 형광등 LED 리폼"
```

---

### Step 2: 보편적 궁금증 나열 (10개 이상)

```
예시: "디밍스위치 + 형광등 → LED 리폼"

1. 디밍스위치에서 LED 설치 가능한가?
2. 왜 다른 업체는 안 된다고 하나?
3. 리폼 vs 전체교체 차이는?
4. 형광등 개수 줄여도 밝기 충분한가?
5. 비용은?
6. 디밍 기능 유지되나?
7. 프레임 상태 안 좋아도 리폼 가능?
8. 시공 시간은?
...
```

---

### Step 3: 3개 선택

**선택 기준:**
1. **이 글에서 답할 수 있는가?** (최우선!)
2. **보편적인가?**
3. **독립적인가?** (억지 연결 X)

```
✅ 선택:
1. 디밍스위치에서 LED 설치 가능한가?
2. 리폼 vs 전체교체?
3. 형광등 9개 → LED 3개면 밝기?

❌ 선택 안 함:
- 비용은? (글에서 구체적 언급 없음)
```

---

### Step 4: 질문 작성

```
✅ 좋은 질문:
"디밍 스위치가 있으면 LED 설치가 안 된다는데 정말인가요?"

❌ 나쁜 질문:
"디밍안정기 고장 났을 때 직접 수리 시도했다가 복잡해진 경험 있으신가요?"
→ 특정 사례, 보편성 없음
```

## 3. 실전 예시

### 예시 1: 디밍스위치 + 형광등 리폼

**입력:**
- 디밍스위치 아파트
- 형광등 9개 → LED 3개
- DIY 실패

**주제:**
```
"디밍스위치 아파트 형광등 LED 리폼"
```

**궁금증 나열:**
```
1. 디밍스위치에서 LED 가능?
2. 리폼 vs 전체교체?
3. 형광등 9개 → LED 3개 밝기?
4. 디밍 기능 유지?
5. 비용?
...
```

**3개 선택 → 질문:**

```
## 디밍 스위치가 있으면 LED 설치가 안 된다는데 정말인가요?

## 등기구 전체를 교체하지 않고 LED 모듈만 바꿀 수 있나요?

## 형광등 9개를 LED 3개로 줄여도 밝기는 충분한가요?
```

---

### 예시 2: 노후 LED 재교체

**입력:**
- LED 10년 사용
- 밝기 약해짐, 색 변화

**주제:**
```
"노후 LED 재교체 필요성"
```

**질문:**

```
## 10년 된 LED조명도 교체가 필요한가요?

## LED 밝기가 약해지고 색이 변했다면 수명이 다한 건가요?

## 이미 LED를 사용하고 있어도 전력을 더 줄일 수 있나요?
```

---

### 예시 3: 형광등 깜박임

**입력:**
- 형광등 깜박임
- 눈 피로

**주제:**
```
"형광등 깜박임 문제 LED로 해결"
```

**질문:**

```
## 형광등이 자꾸 깜박거리는 이유가 뭔가요?

## LED로 바꾸면 깜박임 없이 사용할 수 있나요?

## 형광등처럼 자주 교체하지 않아도 되나요?
```

## 4. 금지 사항

```
❌ 특정 사례를 질문으로
   "디밍안정기 고장 났을 때..."

❌ 억지 스토리 구조
   1번(공감) + 2번(불편) + 3번(해결)

❌ 평서문
   "LED로 바꾸면 밝아집니다."

❌ 판매 냄새
   "저희 업체에서 시공해드릴까요?"
```
