## 1. 전력 절감 표현 10가지

### 📊 표현 패턴표

| 번호 | 표현 패턴 | 사용 예시 |
|------|----------|----------|
| 1 | 소비전력은 A에서 B로 C 줄었습니다 | 소비전력은 432W에서 216W로 216W 줄었습니다 |
| 2 | 전력 소비량이 A에서 B로 C 절감됐습니다 | 전력 소비량이 110W에서 100W로 10W 절감됐습니다 |
| 3 | 기존 A에서 B로 바뀌면서 C를 아끼게 됐습니다 | 기존 144W에서 72W로 바뀌면서 72W를 아끼게 됐습니다 |
| 4 | A를 쓰던 조명이 B만 사용하게 되어 C가 절약됐습니다 | 108W를 쓰던 조명이 50W만 사용하게 되어 58W가 절약됐습니다 |
| 5 | 전력이 A에서 B로 낮아져 C만큼 감소했습니다 | 전력이 432W에서 216W로 낮아져 216W만큼 감소했습니다 |
| 6 | A였던 소비전력이 B로 줄면서 C의 절감 효과를 봤습니다 | 144W였던 소비전력이 72W로 줄면서 72W의 절감 효과를 봤습니다 |
| 7 | 기존 대비 C가 줄어든 B로 전환됐습니다 | 기존 대비 216W가 줄어든 216W로 전환됐습니다 |
| 8 | A 대신 B를 사용하니 C의 전력을 아낄 수 있었습니다 | 432W 대신 216W를 사용하니 216W의 전력을 아낄 수 있었습니다 |
| 9 | 전력 사용량이 A에서 B로 개선되며 C 절약됐습니다 | 전력 사용량이 110W에서 100W로 개선되며 10W 절약됐습니다 |
| 10 | 소비 전력을 A에서 B로 낮춰 C의 효과를 얻었습니다 | 소비 전력을 108W에서 50W로 낮춰 58W의 효과를 얻었습니다 |

### 📝 사용법

**변수 설명:**
- A = 기존 전력 (예: 432W)
- B = 교체 후 전력 (예: 216W)
- C = 절감 전력 (예: 216W)

**적용 원칙:**
- 4개 공간 → 최소 4가지 다른 표현 사용
- 같은 패턴 2회 이상 사용 금지

## 2. 비율 표현 10가지

| 번호 | 표현 | 적용 비율 범위 | 톤 |
|------|------|---------------|-----|
| 1 | 무려 50%나 감소한 거죠 | 40~60% | 강조 |
| 2 | 약 9% 정도 낮아진 셈이에요 | 5~15% | 친근 |
| 3 | 절반 수준으로 떨어진 거예요! | 45~55% | 구어체 |
| 4 | 54% 가까이 줄어든 결과죠 | 50~60% | 정확 |
| 5 | 절반 가량 줄어든 셈입니다 | 40~55% | 격식 |
| 6 | 거의 반토막 난 수준이에요 | 45~55% | 비유 |
| 7 | 50% 이상 개선된 겁니다 | 50% 이상 | 기술 |
| 8 | 절반 이상 낮아졌어요 | 50% 이상 | 친근 |
| 9 | 2분의 1로 줄어든 거죠 | 45~55% | 수학적 |

### 📝 사용법

**선택 기준:**
- 정확한 비율에 맞는 표현 선택
- 톤에 맞춰 순환 사용

## 5. 적용 가이드

### 📋 Step 1: 작성 전 계획

**4개 공간 작성 시 표현 할당:**
```
[거실]
- 전력: 패턴 1번
- 비율: 패턴 1번
- 밝기: 패턴 1번
- 작업: 패턴 1, 3, 5번

[주방]
- 전력: 패턴 2번
- 비율: 패턴 2번
- 밝기: 패턴 2번
- 작업: 패턴 2, 4, 6번

[안방]
- 전력: 패턴 3번
- 비율: 패턴 3번
- 밝기: 패턴 3번
- 작업: 패턴 7, 8, 9번

[작은방]
- 전력: 패턴 4번
- 비율: 패턴 4번
- 밝기: 패턴 4번
- 작업: 패턴 10번
```

---

### 📋 Step 2: 작성 중 참조

**표현이 막힐 때:**

1. **어떤 내용을 쓸지 파악**
   - 전력 절감? → 섹션 1 참조
   - 비율? → 섹션 2 참조
   - 밝기? → 섹션 3 참조
   - 작업 과정? → 섹션 4 참조

2. **해당 표에서 선택**
   - 이미 사용한 패턴 체크
   - 사용하지 않은 패턴 선택

3. **문맥에 맞게 적용**
   - 변수 (A, B, C) 채우기
   - 공간명 맞추기

---

### 📋 Step 3: 작성 후 검토

**Ctrl+F로 반복 검색:**
```
검색어 목록:
- "소비전력은"
- "무려"
- "환해졌습니다"
- "전원을 차단"
- "커버를"
- "철거"
- "장착"
```

**검색 결과:**
- 0~1회: ✅ 좋음
- 2회 이상: ❌ 다른 표현으로 교체

## 7. 실전 팁

### 💡 Tip 1: 문장 종결 다양화
```
❌ 나쁜 예:
소비전력은 432W에서 216W로 216W 줄었습니다.
전력 소비량이 110W에서 100W로 10W 줄었습니다.
기존 144W에서 72W로 72W 줄었습니다.

✅ 좋은 예:
소비전력은 432W에서 216W로 216W 줄었습니다. (격식)
전력 소비량이 110W에서 100W로 10W 절감됐습니다. (중립)
기존 144W에서 72W로 바뀌면서 72W를 아끼게 됐습니다. (친근)
```

---

### 💡 Tip 2: 접속사 활용
```
같은 내용도 접속사로 느낌 바꾸기:

[패턴 A] 그리고 전기세도 절약됩니다.
[패턴 B] 또한 전기세 부담도 줄어들죠.
[패턴 C] 게다가 전력 소비도 낮아집니다.
[패턴 D] 덕분에 전기 요금도 아낄 수 있어요.
```

---

### 💡 Tip 3: 문장 순서 변경
```
[패턴 A] 전력 → 비율
소비전력은 432W에서 216W로 216W 줄었습니다.
무려 50%나 감소한 거죠.

[패턴 B] 비율 → 전력
절반 수준으로 떨어진 거예요!
기존 144W에서 72W로 바뀌면서 72W를 아끼게 됐습니다.
```

---

### 💡 Tip 4: 수치 + 표현 조합
```
[조합 1] 구체적 수치 + 비율 + 감정
216W 줄었습니다 + 무려 50%나 + 감소한 거죠

[조합 2] 비율 + 구체적 수치 + 비유
절반 수준으로 + 72W를 아끼게 + 됐습니다

[조합 3] 비교 + 수치 + 결과
108W를 쓰던 조명이 + 50W만 사용 + 58W가 절약
```

---

### 💡 Tip 5: 공간별 톤 매칭
```
거실 (강조): "정말", "무려", "확실히"
주방 (친근): "~어요", "~죠", "~거든요"
안방 (격식): "~습니다", "~됩니다"
작은방 (구어): "~잖아요", "~더라고요"
```

### 🎯 작업 과정 다양화 예시
```

[안방 - 작업 중]
전원 스위치를 내린 후 시작했습니다. (패턴 4)

기존 PL 형광등을 모두 철거했습니다. (변형)

[작은방 - 작업 중]
안전을 확보한 뒤 작업에 들어갔습니다. (패턴 10)
커버를 분리했습니다. (변형)
기존 PL 형광등을 철거했죠. (변형)
```

## 9. 주의사항

### ⚠️ 주의 1: 기술 용어는 통일
```
✅ 올바른 예:
- LED 모듈 (항상 이렇게)
- 소비전력 (항상 이렇게)
- 형광등 (항상 이렇게)

❌ 잘못된 예:
- LED 모듈 / 엘이디 모듈 / 발광다이오드 모듈 (혼용 금지)
- 소비전력 / 전력소비량 / 사용전력 (혼용 금지)
- 형광등 / 형광램프 (혼용 금지)
```

---

### ⚠️ 주의 2: 수치 표기는 정확히
```
✅ 올바른 예:
- 432W에서 216W로 (정확)
- 110W에서 100W로 (정확)

❌ 잘못된 예:
- 약 430W에서 220W 정도로 (부정확)
- 대략 100W 안팎에서 (모호)
```

**이유:** 유사문서 방지를 위해 표현을 바꾸되, 수치는 정확해야 함

---

### ⚠️ 주의 3: 키워드는 일관성
```
✅ 올바른 예:
- 송도LED조명 (항상 붙여쓰기)
- LED리폼 (항상 붙여쓰기)

❌ 잘못된 예:
- 송도LED조명 / 송도 LED 조명 / 송도엘이디조명
- LED리폼 / LED 리폼 / 엘이디리폼
```

---

### ⚠️ 주의 4: 자연스러움 우선
```
억지로 바꾸지 말 것!

❌ 나쁜 예:
전력량이 432와트에서 216와트로 216와트만큼 줄어들게 됐습니다.
(너무 어색함)

✅ 좋은 예:
소비전력은 432W에서 216W로 216W 줄었습니다.
(자연스럽고 정확함)
```

**원칙:** 다양화 > 자연스러움 (자연스러움이 더 중요!)
