## 2. 독점 기술 핵심 요약 ⭐ 수정

### 🔧 기술 1: 송도 유일 - 디밍스위치 교체 없이 LED 조명 설치

**적용 아파트:**
- 퍼스트월드
- 엑스포 6, 9, 10단지
- 센트럴파크 1차, 2차
- 웰카운티 1, 2, 3, 4단지

---

**문제 (일반 업체):**
```
디밍스위치가 있는 아파트에서 LED 설치 시:
→ 디밍스위치를 일반스위치로 교체 필수
→ 벽면 공사 필요
→ 추가 비용 발생 (스위치 + 공사비)
→ 불편함 4가지:
   1. 추가 비용 부담
   2. 벽면 공사 (도배 훼손)
   3. 시공 시간 증가
   4. 스위치 위치/디자인 변경
```

---

**해결 (휴명):**
```
✅ 디밍스위치 그대로 유지 (교체 불필요)
✅ 벽면 공사 없음 (도배 보존)
✅ 비용 절감 (스위치 + 공사비)
✅ 시공 시간 단축
✅ 기존 스위치 위치/디자인 유지

⚠️ 중요:
- 온/오프만 작동 (켜기/끄기)
- 밝기 조절 기능은 작동 안 함
```

---

**핵심 차별점:**
```
다른 업체: 스위치 교체 + 추가 공사
휴명: 스위치 유지 + 공사 없음

→ 경제적이고 간편한 시공
```

---
